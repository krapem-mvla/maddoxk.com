Elevator Simulation

Elevator Simulation is a realistic and comprehensive simulation tool that provides an interactive and engaging way to explore the inner workings of elevators. Through this project, I explored the workings of a Finite State Machine (FSM) which simulates many complex components of an elevator and how they all interact to create a safe and efficient ride. Elevator Simulation also provides detailed visuals and animations to make the experience as immersive as possible. Learn the basics of elevator operation and gain a better understanding of implementing FSMs.
